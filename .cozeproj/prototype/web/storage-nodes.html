节点接入指南

1. 安装客户端

curl -fsSL https://cloudops.example.com/install.sh | bash

2. 配置节点令牌

cloudops config --token=YOUR_NODE_TOKEN

3. 启动节点服务

cloudops start --mode=storage

完成以上步骤后，节点将自动出现在存储节点列表中。
